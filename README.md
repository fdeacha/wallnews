# Wallnews

Aplicación web full-stack tipo muro de noticias, desarrollada como práctica de programación web. Permite publicar y ver noticias con imágenes, combinando frontend y backend con una base de datos.

## Tecnologías

Backend con Node.js y Express. Vistas renderizadas con EJS junto a HTML y CSS. Base de datos MySQL. Las imágenes subidas se almacenan en la carpeta public/uploads.

## Funcionalidades

Publicación de noticias con título, contenido e imagen. Listado de noticias en un muro. Persistencia de datos en base de datos. Subida y visualización de imágenes.

## Instalación y uso

Cloná el repositorio con git clone https://github.com/fdeacha/wallnews.git, entrá a la carpeta con cd wallnews, instalá las dependencias con npm install y ejecutá node index.js. Luego abrí el navegador en localhost con el puerto configurado en index.js.

## Estructura del proyecto

La carpeta db contiene la configuración de la base de datos, public/uploads guarda las imágenes subidas, views contiene las plantillas EJS, index.js es el punto de entrada de la aplicación y package.json lista las dependencias.
